# Phase 4.2: RFID批量盘点 - 总览

## 概述

实现基于RFID技术的资产批量盘点功能，支持RFID标签管理、读写器适配、批量扫描、数据解析等核心功能。

---

## 1. 业务背景

### 1.1 RFID vs 二维码

| 对比项 | 二维码 | RFID |
|--------|--------|------|
| 读取方式 | 逐个扫描 | 批量读取 |
| 读取距离 | <10cm | 1-10米 |
| 读取速度 | 较慢 | 极快 |
| 标签成本 | 低（几分钱） | 较高（几毛钱） |
| 适用场景 | 少量资产 | 大批量仓库 |

### 1.2 RFID应用场景

- 仓库大批量资产盘点
- 开放区域资产快速盘点
- 资产进出区域自动识别
- 资产位置实时追踪

## 3. RFID标签

### 3.1 标签类型

| 类型 | 频率 | 读取距离 | 适用场景 |
|------|------|----------|----------|
| 低频LF | 125kHz | <10cm | 门禁 |
| 高频HF | 13.56MHz | <1m | 图书馆 |
| 超高频UHF | 860-960MHz | 1-10m | 资产盘点 |

### 3.2 标签数据结构

```
EPC (Electronic Product Code):
- 头部: 协议版本
- 域名管理: 组织代码
- 对象分类: 资产分类
- 序列号: 资产唯一ID

示例: 3034-00001-00001-0000123456
       │    │      │      │
       │    │      │      └─ 资产序列号
       │    │      └───────── 资产分类
       │    └──────────────── 组织代码
       └───────────────────── 协议版本
```

## 4. 读写器适配

### 4.1 支持的读写器

| 品牌 | 型号 | 协议 |
|------|------|------|
| Impinj | R420 | LLRP |
| Alien | ALR-9900 | LLRP |
| 得一微 | DY-R9980 | 自定义协议 |

### 4.2 适配器接口

```python
class RFIDReaderAdapter(ABC):
    @abstractmethod
    def connect(self, host, port) -> bool:
        """连接读写器"""
        pass

    @abstractmethod
    def start_reading(self, callback, duration) -> None:
        """开始读取标签"""
        pass

    @abstractmethod
    def stop_reading(self) -> None:
        """停止读取"""
        pass

    @abstractmethod
    def get_reader_info(self) -> dict:
        """获取读写器信息"""
        pass
```

## 5. 批量扫描流程

### 5.1 区域扫描

```
1. 设置读写器参数
   - 功率: 30dBm
   - 扫描时间: 30秒
   - 区域: A区仓库

2. 开始批量读取
   - 启动读写器
   - 循环读取标签
   - 实时显示进度

3. 数据解析
   - 解析EPC码
   - 匹配资产信息
   - 标记已盘点资产

4. 完成扫描
   - 统计盘点结果
   - 标记未盘点资产
```

## 6. API接口

### 6.1 写入RFID标签

```
POST /api/inventory/rfid/write/
Request: {
    "asset_id": 123,
    "epc": "3034000010000100000123456"
}
```

### 6.2 批量扫描

```
POST /api/inventory/rfid/scan/
Request: {
    "reader_id": 1,
    "duration": 30,
    "task_id": 10
}
Response: {
    "scanned_count": 150,
    "assets": [...]
}
```

### 6.3 扫描进度

```
GET /api/inventory/rfid/scan/{scan_id}/progress/
Response: {
    "status": "running",
    "scanned_count": 50,
    "elapsed": 10
}
```
