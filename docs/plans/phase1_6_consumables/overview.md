# Phase 1.6: 低值易耗品/办公用品管理 - 总览

## 概述

管理低值易耗品和办公用品的入库、领用、库存管理，与固定资产管理形成互补，实现企业物资的全面管理。

---

## 1. 业务背景

### 1.1 管理对象差异

| 对比项 | 固定资产 | 低值易耗品/办公用品 |
|--------|----------|---------------------|
| **价值阈值** | 通常>2000元 | 通常<2000元 |
| **管理颗粒度** | 一物一码 | 批次/库存数量 |
| **折旧处理** | 需要计提折旧 | 不需要折旧 |
| **使用方式** | 领用后可退回 | 领用后消耗 |
| **盘点方式** | 逐一扫码盘点 | 数量盘点/抽盘 |
| **财务处理** | 资本性支出 | 费用性支出 |

### 1.2 核心目标

- 批次管理，快速流转
- 库存预警，及时补货
- 费用统计，成本控制
- 简化流程，提高效率

## 3. 数据模型

### 3.1 ConsumableCategory（耗材分类）

| 字段 | 说明 |
|------|------|
| code | 分类编码 |
| name | 分类名称 |
| parent | 上级分类 |
| unit | 计量单位 |
| min_stock | 最低库存（预警） |
| max_stock | 最高库存 |
| reorder_point | 补货点 |

### 3.2 Consumable（耗材档案）

| 字段 | 说明 |
|------|------|
| code | 耗材编码 |
| name | 耗材名称 |
| category | 所属分类 |
| specification | 规格型号 |
| unit | 计量单位 |
| purchase_price | 参考进价 |
| sale_price | 参考售价 |
| supplier | 默认供应商 |

### 3.3 ConsumableStock（库存台账）

| 字段 | 说明 |
|------|------|
| consumable | 耗材 |
| warehouse | 仓库 |
| quantity | 当前库存 |
| available_qty | 可用库存 |
| in_transit_qty | 在途数量 |

### 3.4 ConsumableTransaction（出入库记录）

| 字段 | 说明 |
|------|------|
| transaction_no | 单据号 |
| transaction_type | 类型（入库/出库） |
| business_type | 业务类型（采购/领用/调拨） |
| consumable | 耗材 |
| quantity | 数量 |
| after_qty | 结存数量 |
| reference | 关联单据 |

## 5. 库存预警

### 预警规则

| 预警级别 | 触发条件 | 处理建议 |
|----------|----------|----------|
| 正常 | 库存 > 补货点 | 无需操作 |
| 需补货 | 库存 ≤ 补货点 | 建议采购 |
| 库存不足 | 库存 ≤ 最低库存 | 紧急采购 |
| 库存超限 | 库存 > 最高库存 | 暂停采购 |

### 预警通知

- 站内消息通知仓库管理员
- 邮件通知采购人员
- 支持群发批量通知

## 6. 业务流程

### 领用流程

```
员工申请
   ↓
部门审批
   ↓
仓库出库
   ↓
库存更新
   ↓
领用确认
```

### 采购流程

```
库存预警触发
   ↓
采购申请
   ↓
审批
   ↓
采购订单
   ↓
入库
   ↓
库存更新
```
