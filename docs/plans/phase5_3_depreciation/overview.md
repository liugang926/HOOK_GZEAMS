# Phase 5.3: 折旧自动计算 - 总览

## 概述

实现固定资产折旧自动计算功能，支持多种折旧方法（直线法、双倍余额递减法、年数总和法），自动生成折旧计提记录。

---

## 1. 业务背景

### 1.1 折旧需求

| 需求 | 说明 |
|------|------|
| **多种方法** | 支持不同折旧方法 |
| **自动计提** | 按月自动计提折旧 |
| **分类处理** | 不同分类不同方法 |
| **财务对接** | 生成财务凭证 |

### 1.2 折旧方法

| 方法 | 适用场景 |
|------|----------|
| 直线法 | 均匀折旧的资产 |
| 双倍余额递减法 | 前期损耗大的资产 |
| 年数总和法 | 前期效益高的资产 |
| 不计提折旧 | 土地、文物等 |

## 3. 折旧方法

### 3.1 直线法

```
月折旧额 = (原值 - 预计净残值) / 使用月数

示例：
原值: 15000元
净残值率: 5%
使用年限: 5年 (60个月)

月折旧额 = (15000 - 15000×5%) / 60 = 237.50元
```

### 3.2 双倍余额递减法

```
月折旧额 = 账面净值 × 2 / 使用年限 / 12

最后两年改为直线法。

示例：
原值: 15000元
使用年限: 5年

第1年折旧 = 15000 × 2/5 = 6000元
第2年折旧 = 9000 × 2/5 = 3600元
...
```

### 3.3 年数总和法

```
月折旧额 = (原值 - 残值) × 剩余月数 / 总月数

总月数 = n × (n + 1) / 2

示例：
使用年限: 5年 (60个月)
总月数 = 60 × 61 / 2 = 1830

第1个月折旧 = (15000-750) × 60 / 1830 = 467.21元
第2个月折旧 = (15000-750) × 59 / 1830 = 459.51元
...
```

## 4. 数据模型

### 4.1 AssetDepreciation（折旧记录）

| 字段 | 说明 |
|------|------|
| asset | 资产 |
| period | 折旧期间（YYYY-MM） |
| depreciation_method | 折旧方法 |
| purchase_price | 原值 |
| residual_value | 残值 |
| useful_life | 使用月数 |
| used_months | 已使用月数 |
| depreciation_amount | 本期折旧额 |
| accumulated_depreciation | 累计折旧 |
| net_value | 资产净值 |
| voucher_status | 凭证状态 |

## 5. 折旧流程

### 5.1 月末计提

```
1. 查询所有应计提折旧的资产
   - 在用状态
   - 已到折旧开始日期
   - 未提足折旧

2. 逐个资产计算折旧
   - 获取折旧方法
   - 计算本期折旧额
   - 更新累计折旧

3. 生成折旧记录
   - 创建折旧记录
   - 生成财务凭证

4. 折旧报表
   - 按分类汇总
   - 按部门汇总
```

### 5.2 变动处理

| 变动类型 | 处理方式 |
|----------|----------|
| 原值变更 | 按新原值重新计算后续折旧 |
| 使用年限变更 | 按新年限重新计算 |
| 报废处置 | 停止计提折旧 |

## 6. API接口

### 6.1 计算折旧

```
POST /api/assets/depreciation/calculate/
Request: {
    "period": "2024-01",
    "asset_ids": "all"  // 或指定ID列表
}
Response: {
    "calculated_count": 500,
    "total_amount": 125000.00
}
```

### 6.2 查询折旧记录

```
GET /api/assets/{id}/depreciations/
Query: {
    "period_from": "2024-01",
    "period_to": "2024-12"
}
Response: {
    "results": [
        {
            "period": "2024-01",
            "amount": 237.50,
            "accumulated": 237.50
        }
    ]
}
```

### 6.3 折旧报表

```
GET /api/assets/depreciation/report/
Query: {
    "period": "2024-01",
    "group_by": "category"
}
Response: {
    "summary": {
        "total_amount": 125000.00,
        "asset_count": 500
    },
    "by_category": [...]
}
```
