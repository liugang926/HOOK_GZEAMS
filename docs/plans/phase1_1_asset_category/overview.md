# Phase 1.1: 资产分类体系 - 总览

## 概述

建立完整的固定资产分类体系，支持树形多级分类、折旧配置、分类统计等功能，是资产管理的基石。

---

## 1. 业务背景

### 1.1 功能目标

- 建立符合国家标准的资产分类体系
- 支持自定义分类扩展
- 关联折旧方法和使用年限
- 支持分类维度的资产统计

### 1.2 核心需求

| 需求 | 说明 |
|------|------|
| **标准分类** | 参考国标分类，预设常用分类 |
| **多级分类** | 支持至少3级分类树形结构 |
| **折旧配置** | 每个分类可配置默认折旧方法 |
| **分类统计** | 按分类统计资产数量和价值 |

## 3. 数据模型

### 3.1 AssetCategory（资产分类）

| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 分类编码（唯一） |
| name | string | 分类名称 |
| parent | FK | 上级分类 |
| depreciation_method | string | 折旧方法 |
| default_useful_life | int | 默认使用年限(月) |
| residual_rate | decimal | 预计净残值率(%) |
| is_custom | bool | 是否自定义分类 |
| sort_order | int | 排序号 |
| is_active | bool | 是否启用 |

---

## 4. 折旧方法

| 方法 | 代码 | 公式 | 适用场景 |
|------|------|------|----------|
| 直线法 | straight_line | (原值-残值)/使用月数 | 均匀折旧资产 |
| 双倍余额递减法 | double_declining | 账面净值×2/使用年限/12 | 前期多折旧 |
| 年数总和法 | sum_of_years | (原值-残值)×剩余月数/总月数 | 前期多折旧 |
| 不计提折旧 | no_depreciation | - | 土地等 |
