# Phase 6: User Portal (用户门户) - 总览

## 概述

用户门户是企业固定资产管理系统中面向普通员工的核心入口，提供个人资产管理、申请记录查询、待办事项处理等功能。

---

## 1. 业务背景

### 1.1 目标用户

| 用户类型 | 需求 |
|----------|------|
| **普通员工** | 查看我的资产、查询申请记录 |
| **部门负责人** | 审批资产申请、查看部门资产 |
| **资产管理员** | 处理资产退库、资产借用 |

### 1.2 核心功能

- 我的资产：查看保管/使用的资产
- 我的申请：领用/调拨/借用申请记录
- 我的待办：待审批/待处理事项
- 资产退库：退库申请
- 资产借用：临时借用

## 3. 我的资产

### 3.1 资产分类

| 分类 | 说明 |
|------|------|
| **保管资产** | 资产保管人为当前用户 |
| **使用资产** | 资产使用人为当前用户 |
| **借用资产** | 临时借用的资产 |

### 3.2 资产信息

| 字段 | 说明 |
|------|------|
| 资产编码 | 唯一标识 |
| 资产名称 | 资产名称 |
| 资产分类 | 所属分类 |
| 领用日期 | 开始使用日期 |
| 存放位置 | 当前位置 |

## 4. 我的申请

### 4.1 申请类型

| 类型 | 说明 |
|------|------|
| **领用申请** | 资产领用 |
| **调拨申请** | 部门间调拨 |
| **借用申请** | 临时借用 |
| **退库申请** | 资产归还 |

### 4.2 申请状态

| 状态 | 说明 |
|------|------|
| 草稿 | 未提交 |
| 待审批 | 等待审批 |
| 已批准 | 审批通过 |
| 已拒绝 | 审批拒绝 |
| 已完成 | 流程完成 |

## 6. API接口

### 6.1 我的资产

```
GET /api/portal/my-assets/
Query: {
    "category": "custody"  // custody/using/borrowing
}
Response: {
    "results": [
        {
            "asset_code": "ZC001",
            "asset_name": "MacBook Pro",
            "pickup_date": "2024-01-01"
        }
    ]
}
```

### 6.2 我的申请

```
GET /api/portal/my-applications/
Query: {
    "type": "pickup",
    "status": "pending"
}
```

```
}
```
