# Phase 4.4: 用户端资产信息展示设计

## 概述

本文档详细描述盘点任务执行过程中，用户端（移动端）如何展示资产信息，包括扫码后的详情展示、跨任务汇总视图，以及管理员自定义字段配置能力。

---

## 1. 扫码后资产信息展示（滑动卡片）

### 1.1 设计原则

用户扫码确认资产后，通过**水平滑动的卡片组**展示资产完整信息，解决移动端屏幕有限与信息量大的矛盾。

### 1.2 卡片分组结构

| 卡片名称 | 默认显示 | 包含字段 |
|---------|---------|---------|
| 基础信息卡 | 是 | 资产照片、资产编码、资产名称、分类、规格型号 |
| 使用信息卡 | 否 | 存放地点、保管人、使用状态、领用日期 |
| 财务信息卡 | 否 | 资产原值、累计折旧、净值、采购日期、供应商 |
| 盘点快照卡 | 否 | 快照生成时间、上次盘点时间、上次盘点人、历史记录 |

### 1.4 各卡片详细字段

#### 基础信息卡（默认显示）

| 字段 | 显示规则 | 说明 |
|------|---------|------|
| 资产照片 | 大图显示 | 支持点击放大、左右滑动查看多张照片 |
| 资产编码 | 高亮显示 | 便于扫码后快速核对是否扫描正确 |
| 资产名称 | 正常显示 | - |
| 资产分类 | 层级显示 | 显示完整分类路径，如：电子设备 > 电脑 > 笔记本 |
| 规格型号 | 正常显示 | 支持长按复制 |

#### 使用信息卡

| 字段 | 显示规则 | 说明 |
|------|---------|------|
| 存放地点 | 对比显示 | 快照位置 vs 当前实际位置（如有变化则高亮） |
| 保管人 | 卡片式显示 | 姓名 + 部门 + 头像 |
| 使用状态 | 标签式显示 | 在用(绿)/闲置(灰)/维修中(橙)/待报废(红) |
| 领用日期 | 日期格式 | YYYY-MM-DD |

#### 财务信息卡

| 字段 | 显示规则 | 说明 |
|------|---------|------|
| 资产原值 | 货币格式 | ¥12,999.00 |
| 累计折旧 | 货币格式 | ¥4,332.15 |
| 净值 | 货币格式 | ¥8,666.85 |
| 采购日期 | 日期格式 | YYYY-MM-DD |
| 供应商 | 文本显示 | - |

#### 盘点快照卡

| 字段 | 显示规则 | 说明 |
|------|---------|------|
| 快照生成时间 | 时间戳 | 显示相对时间（如：2小时前生成） |
| 快照时状态 | 标签式 | 与当时的使用状态一致 |
| 上次盘点时间 | 日期 | 便于判断盘点频率 |
| 上次盘点人 | 用户信息 | 姓名 + 盘点时间 |
| 历史记录 | 摘要列表 | 最近3次盘点记录简述 |

### 1.5 交互细节

1. **卡片切换**：左右滑动或点击顶部标签切换
2. **照片放大**：点击资产照片进入全屏查看模式，支持缩放
3. **字段长按**：支持长按复制文本内容（编码、型号等）
4. **状态反馈**：位置不一致时，使用信息卡中的位置字段高亮显示

## 3. 管理员自定义字段配置

### 3.1 配置层级

字段配置支持三层结构：

| 层级 | 说明 | 优先级 |
|------|------|--------|
| 系统默认 | 系统预设的字段配置 | 最低 |
| 组织模板 | 各组织可保存自己的配置模板 | 中 |
| 任务级 | 单个任务可覆盖模板配置 | 最高 |

### 3.2 配置能力

#### 3.2.1 字段选择

管理员可勾选/取消勾选列表中显示的字段：

## 4. 数据获取策略（混合模式）

### 4.1 任务下发时预取

任务分配给用户时，自动预取以下基础信息到本地：

| 数据类型 | 存储位置 | 用途 |
|---------|---------|------|
| 资产编码 | IndexedDB | 扫码后立即匹配 |
| 资产名称 | IndexedDB | 快速显示 |
| 资产照片URL | IndexedDB | 预加载缩略图 |
| 存放地点 | IndexedDB | 快照位置对比 |
| 保管人 | IndexedDB | 显示保管人信息 |
| 资产分类 | IndexedDB | 分组展示 |

### 4.2 扫码后补充获取

扫码确认后，实时从服务器获取：

| 数据类型 | 获取时机 | 用途 |
|---------|---------|------|
| 最新资产状态 | 扫码后 | 确保状态准确性 |
| 详细规格型号 | 点击详情卡时 | 按需加载 |
| 财务信息 | 点击财务卡时 | 按需加载 |
| 当前实际位置 | 扫码后 | 位置变更对比 |
| 高清资产照片 | 点击放大时 | 节省流量 |

### 4.3 离线处理逻辑

```
┌─────────────────────────────────────────────────────────────┐
│  离线模式处理流程                                            │
└─────────────────────────────────────────────────────────────┘

  任务下发时
       │
       ▼
  ┌─────────────────┐
  │ 预取基础信息     │
  │ 存入IndexedDB   │
  └─────────────────┘
       │
       ▼
  ┌─────────────────┐    有网络    ┌─────────────────┐
  │ 用户扫码盘点     │ ──────────> │ 实时获取详细信息 │
  └─────────────────┘              └─────────────────┘
       │                                     │
       │ 无网络                              │
       ▼                                     ▼
  ┌─────────────────┐              ┌─────────────────┐
  │ 使用本地缓存     │ <─────────── │ 更新本地缓存     │
  │ 显示基础信息     │    网络恢复   │ 后台同步最新数据 │
  └─────────────────┘              └─────────────────┘
       │

  ┌─────────────────┐
  │ 盘点记录暂存     │
  │ 等待网络恢复     │
  └─────────────────┘
```
