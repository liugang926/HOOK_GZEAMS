# Phase 5.2: 财务凭证集成 - 总览

## 概述

基于通用集成框架实现财务凭证集成，支持将资产相关业务自动生成财务凭证并推送到ERP财务系统。

---

## 1. 业务背景

### 1.1 凭证类型

| 业务类型 | 凭证类型 | 说明 |
|----------|----------|------|
| 资产购入 | 记账凭证 | 借：固定资产 贷：银行存款 |
| 资产折旧 | 记账凭证 | 借：折旧费用 贷：累计折旧 |
| 资产处置 | 记账凭证 | 固定资产清理 |
| 资产调拨 | 转账凭证 | 内部调拨 |

### 1.2 集成价值

- 自动生成凭证，减少人工操作
- 保证账务一致性
- 满足审计要求

## 3. 数据模型

### 3.1 VoucherTemplate（凭证模板）

| 字段 | 说明 |
|------|------|
| business_type | 业务类型 |
| voucher_type | 凭证类型（记/收/付/转） |
| template_name | 模板名称 |
| entry_templates | 分录模板(JSON) |

### 3.2 分录模板结构

```json
{
    "entry_templates": [
        {
            "direction": "debit",
            "account_code": "1601",
            "account_name": "固定资产",
            "amount_formula": "purchase_price",
            "description": "购入{{asset_name}}"
        },
        {
            "direction": "credit",
            "account_code": "1002",
            "account_name": "银行存款",
            "amount_formula": "purchase_price",
            "description": "支付{{supplier_name}}货款"
        }
    ]
}
```

### 3.3 Voucher（凭证记录）

| 字段 | 说明 |
|------|------|
| voucher_no | 凭证号 |
| business_type | 业务类型 |
| business_id | 业务单据ID |
| voucher_date | 凭证日期 |
| entry_details | 分录明细(JSON) |
| erp_voucher_no | ERP凭证号 |
| push_status | 推送状态 |

## 4. 凭证生成规则

### 4.1 资产购入凭证

```
借：固定资产          15000
借：应交税费-进项税  1950
    贷：应付账款/银行存款  16950
```

### 4.2 资产折旧凭证

```
借：管理费用-折旧费    250
    贷：累计折旧         250
```

### 4.3 资产处置凭证

```
1. 转入清理：
    借：固定资产清理  5000
    借：累计折旧      10000
        贷：固定资产    15000

2. 收到价款：
    借：银行存款      3000
        贷：固定资产清理 3000

3. 确认损益：
    借：营业外支出    2000
        贷：固定资产清理 2000
```

## 6. API接口

### 6.1 生成凭证

```
POST /api/finance/vouchers/generate/
Request: {
    "business_type": "asset_purchase",
    "business_id": 100
}
Response: {
    "voucher_id": 1000,
    "voucher_no": "PZ20240115001",
    "entry_count": 3
}
```

### 6.2 推送ERP

```
POST /api/finance/vouchers/{id}/push/
Response: {
    "success": true,
    "erp_voucher_no": "ERP001"
}
```

### 6.3 查询凭证

```
GET /api/finance/vouchers/
Query: {
    "business_type": "asset_purchase",
    "date_from": "2024-01-01"
}
```
