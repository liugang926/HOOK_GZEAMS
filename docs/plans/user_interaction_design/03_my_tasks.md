# 03. 我的待办模块 - 详细设计

## 概述

"我的待办"模块汇总用户需要处理的所有待办事项，包括审批任务、盘点任务、资产领取确认等。本设计实现多维度分组、快速操作面板、移动端滑动操作等功能。

---

## 1. 多维度分组视图

### 1.1 待办类型

| 类型编码 | 类型名称 | 来源 | 操作 |
|---------|---------|------|------|
| approval | 审批待办 | 工作流引擎 | 同意/拒绝/转审 |
| inventory | 盘点任务 | 盘点模块 | 开始盘点 |
| distribute | 发放确认 | 领用模块 | 确认发放 |
| receive | 接收确认 | 调拨模块 | 确认接收 |
| return_confirm | 归还确认 | 借用模块 | 确认归还 |
| remind | 提醒通知 | 通知模块 | 标记已读 |

| 维度 | 说明 | 分组项 |

| **按待办类型** | 按业务类型分组 | 审批、盘点、发放、接收、归还 |

## 2. 快速操作面板

### 2.1 PC端快速操作

```
┌─────────────────────────────────────────────────────────────────┐
│  批量操作                          已选择 3 项                   │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐  │
│  │ 批量通过 │ │ 批量拒绝 │ │ 批量转审 │ │ 标记完成 │ │ 导出到期 │  │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────┘  │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │ 批量审批评论:                                                ││
```

### 2.2 快速操作说明

| 操作 | 适用类型 | 说明 |
|------|---------|------|
| 批量通过 | 审批待办 | 一键通过选中的待审批申请 |
| 批量拒绝 | 审批待办 | 一键拒绝选中的待审批申请，需填写原因 |
| 批量转审 | 审批待办 | 将选中待办转给其他人处理 |
| 标记完成 | 盘点/发放 | 标记选中待办为已完成 |
| 导出到期 | 全部 | 导出即将到期的待办清单 |

### 5.2 批量操作

```http
POST /api/portal/my-tasks/batch-action/

Request:
{
  "action": "approve",         # approve | reject | transfer | complete
  "task_ids": ["task_001", "task_002", "task_003"],
  "comment": "同意"
}

Response:
{
  "success": true,
  "processed": 3,
  "failed": 0,
  "results": [
    { "task_id": "task_001", "success": true },
    { "task_id": "task_002", "success": true },
    { "task_id": "task_003", "success": true }
  ]
}
```

### 5.3 转审操作

```http
POST /api/portal/my-tasks/{task_id}/transfer/

Request:
{
  "to_user_id": 5,
  "comment": "转由王工审批"
}

Response:
{
  "success": true,
  "message": "待办已转给王工"
}
```

## 6. 待办提醒

### 6.1 提醒规则

| 规则 | 触发条件 | 提醒方式 |
|------|---------|---------|
| 即时提醒 | 新待办分配 | 推送通知 |
| 逾期提醒 | 待办已逾期 | 推送 + 短信 |
| 临期提醒 | 截止前1天 | 推送通知 |
| 逾期升级 | 逾期超过3天 | 推送 + 短信 + 通知上级 |
