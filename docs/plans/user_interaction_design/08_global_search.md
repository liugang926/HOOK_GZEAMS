# 08. 全局搜索功能 - 详细设计

## 概述

全局搜索功能允许用户在系统中快速搜索资产、申请、待办、用户等各类信息，支持分类结果展示、搜索历史、智能建议等。

### 4.2 搜索快捷命令

| 命令 | 说明 | 示例 |
|------|------|------|
| type: | 限制搜索类型 | `type:asset MacBook` 只搜索资产 |
| status: | 限制状态 | `status:in_use 显示器` 只搜索在用资产 |
| dept: | 限制部门 | `dept:技术部 张三` 只搜索技术部 |
| after: | 时间范围之后 | `after:2024-01-01` 搜索1月1日后的内容 |
| before: | 时间范围之前 | `before:2024-01-31` 搜索1月31日前的内容 |

## 5. API设计

### 5.1 搜索接口

```http
GET /api/search/

Query Parameters:
  - q: string               # 搜索关键词
  - types: list             # 搜索类型 (asset | request | task | user)
  - filters: dict           # 筛选条件
  - page: integer
  - page_size: integer

Response:
{
  "query": "MacBook",
  "total": 15,
  "took": 23,
  "results": {
    "assets": {
      "total": 5,
      "items": [
        {
          "type": "asset",
          "id": "asset_001",
          "asset_code": "ZC001",
          "asset_name": "MacBook Pro 16寸",
          "highlight": "MacBook <em>Pro</em> 16寸",
          "category": "笔记本电脑",
          "status": "in_use",
          "custodian": "张三",
          "department": "技术部"
        }
      ]
    },
    "requests": {
      "total": 8,
      "items": [...]
    },
    "users": {
      "total": 2,
      "items": [...]
    }
  }
}
```

### 5.2 搜索建议接口

```http
GET /api/search/suggest/

Query Parameters:
  - q: string               # 输入的前缀

Response:
{
  "query": "Mac",
  "suggestions": [
    { "text": "MacBook Pro", "type": "keyword", "count": 128 },
    { "text": "MacBook Air", "type": "keyword", "count": 45 },
    { "text": "MacBook Pro 16寸", "type": "asset", "id": "asset_001" }
  ]
}
```

### 5.3 搜索历史接口

```http
GET /api/search/history/

Response:
{
  "history": [
    { "query": "MacBook Pro", "searched_at": "2024-01-10T10:30:00Z" },
    { "query": "显示器", "searched_at": "2024-01-10T09:15:00Z" }
  ],
  "popular": [
    { "query": "MacBook", "count": 256 },
    { "query": "显示器", "count": 128 }
  ]
}

POST /api/search/history/

Request:
{
  "query": "MacBook Pro"
}

DELETE /api/search/history/{id}/
DELETE /api/search/history/
```
