# 用户端前后端交互设计总览

## 概述

本文档是对 HOOK GZEAMS 系统中所有用户端前后端交互模块的统一设计规范，基于盘点任务资产信息展示的设计模式，确保全系统交互体验的一致性。

---

## 设计原则

### 1. 一致性原则
- **列表展示**：所有列表类页面支持多维度分组、字段配置
- **详情展示**：移动端统一使用滑动卡片模式
- **操作入口**：统一的快捷操作、更多操作入口

### 2. 渐进式展示
- **默认简洁**：默认只显示核心信息
- **按需展开**：详细信息通过点击、滑动逐步展示
- **智能预加载**：基础信息预取，详细内容按需加载

### 3. 响应式适配
- **PC端**：充分利用屏幕空间，表格/卡片混合布局
- **移动端**：卡片式、滑动交互优先
- **平板端**：自动适配布局

### 4. 离线优先
- **数据预取**：任务/场景启动时预取基础数据
- **本地缓存**：IndexedDB 存储常用数据
- **智能同步**：网络恢复时自动同步

## 通用设计模式

### 模式1：多维度分组视图

适用于所有列表类页面（我的资产、我的申请、我的待办等）

```
┌─────────────────────────────────────────────────────────────┐
│  [按关系] [按状态] [按部门] [按地点] [按分类]              │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │  📦 保管中                                  50项  80%│   │
│  │  ┌─────────────────────────────────────────────┐   │   │
│  │  │ 📦 ZC001  MacBook Pro          ✅ 在用        │   │   │
│  │  │ 📦 ZC002  Dell显示器            ✅ 在用        │   │   │
│  │  │ ... [展开全部]                               │   │   │
│  │  └─────────────────────────────────────────────┘   │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

**支持维度：**
- 我的资产：关系类型、资产状态、使用部门、存放地点、资产分类
- 我的申请：申请类型、审批状态、时间跨度、申请部门
- 我的待办：待办类型、优先级、截止时间、来源模块

适用于所有移动端详情展示

│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │

---

### 模式3：字段配置系统

适用于所有列表类页面

**配置层级：**
```
系统默认配置 (最低优先级)
    ↓
组织级配置模板 (管理员配置，全组织生效)
    ↓
模块级配置模板 (针对特定模块的配置)
    ↓
用户个性化配置 (用户自定义，最高优先级)
```

**配置能力：**
- 字段选择：勾选/取消显示字段
- 字段排序：拖拽调整顺序
- 字段宽度：拖拽调整列宽
- 对齐方式：左对齐/居中/右对齐
- 条件格式：根据字段值设置颜色高亮

---

### 模式4：混合数据获取

适用于所有需要离线支持的场景

```
任务下发/页面加载
       │
       ▼
  ┌─────────────────┐
  │ 预取基础信息     │
  │ (IndexedDB)     │
  └─────────────────┘
       │
       ├──── 有网络 ────> 实时获取详细信息
       │                      │
       │                      ▼
       │              ┌─────────────────┐
       │              │ 更新本地缓存     │
       │              └─────────────────┘
       │
       └──── 无网络 ────> 使用本地缓存
                              │
                              ▼
                       ┌─────────────────┐
                       │ 操作记录暂存     │
                       │ 等待网络恢复     │
                       └─────────────────┘
```

**预取数据时机：**
- 盘点任务分配：预取资产基础信息
- 移动端首页加载：预取待办、资产概览
- 扫码页面打开：预取常用资产信息
- 离线模式进入：全量预取用户数据

## API设计规范

### 列表查询API

所有列表类查询遵循统一格式：

```http
GET /api/{module}/list/

Query Parameters:
  - group_by: string          # 分组维度
  - group_id: string          # 分组ID（获取单个分组内容时）
  - filters: dict             # 筛选条件
  - search: string            # 搜索关键词
  - sort_by: string           # 排序字段
  - sort_order: string        # 排序方向 (asc/desc)
  - page: integer             # 页码
  - page_size: integer        # 每页数量

Response:
{
  "total": 150,
  "group_by": "relation_type",
  "groups": [
    {
      "group_id": "custodian",
      "group_name": "保管中",
      "group_icon": "📦",
      "group_color": "#67C23A",
      "total_count": 50,
      "sub_counts": {
        "in_use": 40,
        "idle": 10
      },
      "items": [...]
    }
  ]
}
```

### 详情获取API

详情获取支持渐进式加载：

```http
GET /api/{module}/{id}/detail/

Query Parameters:
  - sections: list           # 需要返回的详情区块
                              # ["basic", "usage", "financial", "history"]

Response:
{
  "basic": {
    "asset_code": "ZC001",
    "asset_name": "MacBook Pro",
    ...
  },
  "usage": {
    "location": "3楼A区",
    "custodian": {...},
    ...
  },
  "_meta": {
    "available_sections": ["basic", "usage", "financial", "history"],
    "loaded_sections": ["basic", "usage"]
  }
}
```

### 字段配置API

```http
GET /api/system/field-templates/

Response:
{
  "templates": [
    {
      "id": "tpl_my_assets_default",
      "name": "我的资产-默认",
      "module": "my_assets",
      "scope": "organization",
      "config": {
        "list_fields": [...],
        "detail_cards": [...]
      }
    }
  ]
}

POST /api/system/field-templates/
PUT /api/system/field-templates/{id}/
DELETE /api/system/field-templates/{id}/
POST /api/system/field-templates/{id}/set-default/
```

## 前端组件规范

### 通用分组列表组件

```vue
<GroupedListView
  :module="module"
  :group-by.sync="groupBy"
  :groups="groups"
  :loading="loading"
  @group-click="handleGroupClick"
  @item-click="handleItemClick"
>
  <template #group-header="{ group }">
    <!-- 自定义分组头部 -->
  </template>

  <template #item="{ item }">
    <!-- 自定义列表项 -->
  </template>
</GroupedListView>
```

### 通用滑动卡片组件

```vue
<SwipeCardView
  :cards="cards"
  :active-card.sync="activeCard"
  @card-change="handleCardChange"
>
  <template #card-basic="{ data }">
    <!-- 基础信息卡片内容 -->
  </template>

  <template #card-usage="{ data }">
    <!-- 使用信息卡片内容 -->
  </template>

  <template #card-financial="{ data }">
    <!-- 财务信息卡片内容 -->
  </template>
</SwipeCardView>
```

### 字段配置组件

```vue
<FieldConfigPanel
  v-model="config"
  :module="module"
  :view-type="viewType"
  :available-fields="availableFields"
  @save="handleSaveConfig"
  @reset="handleResetConfig"
/>
```
