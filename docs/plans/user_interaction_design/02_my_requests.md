# 02. 我的申请模块 - 详细设计

## 概述

"我的申请"模块汇总用户发起的所有业务申请，包括资产领用、调拨、退库、借用，以及耗材领用、耗材采购等。本设计实现跨类型汇总统计、智能分组视图、移动端卡片优化等功能。

---

## 1. 跨类型汇总统计

### 1.1 申请类型

| 类型编码 | 类型名称 | 所属模块 | 状态流转 |
|---------|---------|---------|---------|
| pickup | 资产领用 | 资产管理 | 草稿 → 待审批 → 已批准 → 已完成 |
| loan | 资产借用 | 资产管理 | 草稿 → 待审批 → 借用中 → 已归还 |
| transfer | 资产调拨 | 资产管理 | 草稿 → 调出审批 → 调入审批 → 已完成 |
| return | 资产退库 | 资产管理 | 草稿 → 待审批 → 已完成 |
| consumable_issue | 耗材领用 | 耗材管理 | 草稿 → 待审批 → 已发放 |
| consumable_purchase | 耗材采购 | 耗材管理 | 草稿 → 待审批 → 采购中 → 已完成 |

## 2. 智能分组视图

### 2.1 支持的分组维度

| 维度 | 说明 | 分组项 |
|------|------|--------|
| **按申请类型** | 按业务类型分组 | 领用、借用、调拨、退库、耗材领用、耗材采购 |
| **按审批状态** | 按审批进度分组 | 草稿、待审批、已批准、已拒绝、已完成 |
| **按时间跨度** | 按申请时间分组 | 今天、本周、本月、更早 |
| **按处理时效** | 按处理速度分组 | 即时完成、正常处理、处理超时 |
| **按申请部门** | 按申请部门分组 | 技术部、财务部、行政部 |

## 3. 字段配置能力

### 3.1 可配置字段清单

#### PC端列表可配置字段

| 字段 | 默认显示 | 说明 |
|------|---------|------|
| 单号 | ✅ | 主键，不可隐藏 |
| 申请类型 | ✅ | 根据分组维度自动调整 |
| 摘要 | ✅ | 申请内容摘要 |
| 申请资产/耗材 | ✅ | 涉及的资产或耗材 |
| 审批状态 | ✅ | 当前审批状态 |
| 当前节点 | ✅ | 审批流转到的节点 |
| 申请时间 | ✅ | 创建时间 |
| 申请人 | ✅ | 可隐藏（自己申请时） |
| 申请部门 | ✅ | 可隐藏 |
| 预计完成时间 | ❌ | 可显示 |
| 实际完成时间 | ❌ | 可显示 |
| 处理时长 | ❌ | 可显示 |
| 操作 | ✅ | 固定在右侧 |

#### 移动端卡片可配置字段

| 优先级 | 字段 | 说明 |
|-------|------|------|
| P0（始终显示） | 单号、类型图标、摘要、状态标签 | 核心信息 |
| P1（默认显示） | 申请时间、涉及资产简要、当前节点 | 关键信息 |
| P2（点击展开） | 申请人、部门、预计完成时间 | 次要信息 |
| P3（详情页） | 审批历史、操作日志、备注 | 详细信息 |

### 3.2 字段配置模板

| 模板名称 | 说明 | 配置特点 |
|---------|------|---------|
| 默认模板 | 标准显示 | 单号、类型、摘要、状态、时间 |
| 跟进模板 | 便于跟进审批进度 | 增加当前节点、催办按钮 |
| 统计模板 | 便于统计时长 | 增加处理时长、完成时间 |
| 精简模板 | 快速浏览 | 只显示单号、摘要、状态 |

## 7. 特殊功能设计

### 7.1 催办功能

**催办规则：**
- 每个申请最多可催办3次
- 两次催办间隔至少2小时
- 超过审批时限的申请可免限制催办

**催办通知：**
- 向当前审批节点发送系统通知
- 可选抄送申请人上级
- 记录催办历史

### 7.2 撤回功能

**撤回条件：**
- 仅草稿、待审批状态可撤回
- 已批准的申请不可撤回（需走拒绝流程）
- 撤回后申请变为草稿状态

**撤回确认：**
```
┌─────────────────────────────────────────┐
│  确认撤回申请                           │
├─────────────────────────────────────────┤
│                                         │
│  您确定要撤回此申请吗？                 │
│                                         │
│  单号: LY202401001                      │
│  类型: 资产领用                          │
│  当前状态: 待审批 - 部门审批             │
│                                         │
│  撤回原因（可选）:                       │
│  ┌─────────────────────────────────────┐│
│  │                                     ││
│  └─────────────────────────────────────┘│
│                                         │
│           [取消]  [确认撤回]             │
└─────────────────────────────────────────┘
```
