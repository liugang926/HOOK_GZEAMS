# 09. 个人中心模块 - 详细设计

## 概述

个人中心模块允许用户查看和编辑个人信息、设置偏好、管理通知偏好、切换主部门等。

## 4. API设计

### 4.1 获取个人信息

```http
GET /api/portal/profile/

Response:
{
  "user": {
    "id": 1,
    "username": "zhangsan",
    "real_name": "张三",
    "employee_no": "E001234",
    "email": "zhangsan@company.com",
    "phone": "13800123456",
    "avatar": "https://...",
    "primary_department": {
      "id": 5,
      "name": "技术部",
      "path": "总部 > 技术部 > 研发组"
    },
    "departments": [
      { "id": 5, "name": "技术部", "is_primary": true },
      { "id": 8, "name": "行政部", "is_primary": false }
    ],
    "roles": ["普通用户"],
    "permissions": ["asset:pickup", "inventory:execute"]
  },
  "statistics": {
    "assets_count": 100,
    "requests_count": 56,
    "tasks_pending": 12
  }
}
```

### 4.2 更新个人信息

```http
PUT /api/portal/profile/

Request:
{
  "real_name": "张三",
  "email": "zhangsan@company.com",
  "phone": "13800123456",
  "avatar": "base64..."
}

Response:
{
  "success": true,
  "message": "个人信息已更新"
}
```

### 4.3 切换主部门

```http
POST /api/portal/profile/switch-department/

Request:
{
  "department_id": 8
}

Response:
{
  "success": true,
  "message": "已切换到行政部",
  "new_department": {
    "id": 8,
    "name": "行政部"
  }
}
```

### 4.4 更新偏好设置

```http
PUT /api/portal/profile/preferences/

Request:
{
  "display": {
    "my_assets_template": "simple",
    "my_assets_sort_by": "asset_code",
    "my_assets_sort_order": "asc",
    "page_size": 20
  },
  "notification": {
    "push_enabled": true,
    "email_enabled": true,
    "sms_enabled": false,
    "quiet_hours": {
      "enabled": true,
      "start": "22:00",
      "end": "08:00"
    }
  }
}
```
