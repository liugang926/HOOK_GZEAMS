# 07. 工作流审批模块 - 详细设计

## 概述

工作流审批模块汇聚用户需要处理的所有审批事项（资产领用、调拨、借用、耗材申请等），提供统一审批入口、批量审批、审批历史追溯等功能。

## 4. API设计

### 4.1 获取待审批列表

```http
GET /api/workflow/my-approvals/

Query Parameters:
  - type: string             # 申请类型筛选
  - status: string           # 状态筛选
  - page: integer
  - page_size: integer

Response:
{
  "summary": {
    "total_count": 25,
    "pending_count": 12,
    "approved_count": 8,
    "rejected_count": 3,
    "transferred_count": 2
  },
  "items": [
    {
      "id": "approval_001",
      "request_type": "pickup",
      "request_type_label": "资产领用",
      "request_no": "LY202401001",
      "summary": "办公电脑申请",
      "applicant": {
        "id": 1,
        "name": "张三",
        "department": "技术部",
        "avatar": "https://..."
      },
      "items_count": 2,
      "items": [
        {
          "asset_code": "ZC001",
          "asset_name": "MacBook Pro 16寸",
          "quantity": 1
        }
      ],
      "current_node": "department_approval",
      "current_node_label": "部门审批",
      "created_at": "2024-01-10T10:30:00Z",
      "due_date": "2024-01-17T18:00:00Z",
      "is_overdue": false
    }
  ]
}
```

### 4.2 审批操作

```http
POST /api/workflow/my-approvals/{approval_id}/approve/

Request:
{
  "decision": "approved",     # approved | rejected | transferred
  "comment": "同意",
  "transfer_to": null          # 转审时必填
}

Response:
{
  "success": true,
  "message": "审批完成",
  "next_node": {
    "id": "asset_approval",
    "name": "资产审批",
    "assignees": [...]
  }
}
```

### 4.3 批量审批

```http
POST /api/workflow/my-approvals/batch-approve/

Request:
{
  "approval_ids": ["approval_001", "approval_002"],
  "decision": "approved",
  "comment": "同意"
}

Response:
{
  "success": true,
  "processed": 2,
  "failed": 0
}
```
