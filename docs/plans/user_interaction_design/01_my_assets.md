# 01. 我的资产模块 - 详细设计

## 概述

"我的资产"模块是用户门户的核心功能，展示用户与资产的所有关联关系（保管、借用、领用）。本设计基于盘点任务资产信息展示的设计模式，实现多维度分组、字段配置、滑动卡片详情等功能。

---

## 1. 多维度分组视图

### 1.1 支持的分组维度

| 维度 | 说明 | 分组项示例 |
|------|------|-----------|
| **按关系类型** | 按用户与资产的关系分组 | 保管中、借用中、领用、待接收 |
| **按资产状态** | 按资产当前使用状态分组 | 在用、闲置、维修中、待报废 |
| **按使用部门** | 按资产所属部门分组 | 技术部、财务部、行政部 |
| **按存放地点** | 按资产存放位置分组 | 3楼A区、3楼B区、会议室 |
| **按资产分类** | 按资产分类层级分组 | 电子设备、办公家具、车辆 |

## 2. 字段配置能力

### 2.1 可配置字段清单

#### PC端列表可配置字段

| 字段 | 默认显示 | 说明 |
|------|---------|------|
| 资产编码 | ✅ | 主键，不可隐藏 |
| 资产名称 | ✅ | 主键，不可隐藏 |
| 资产分类 | ✅ | 可隐藏 |
| 规格型号 | ✅ | 可隐藏 |
| 资产照片 | ❌ | 可显示 |
| 关系类型 | ✅ | 根据分组维度自动调整 |
| 资产状态 | ✅ | 可隐藏 |
| 使用部门 | ✅ | 可隐藏 |
| 存放地点 | ✅ | 可隐藏 |
| 保管人 | ✅ | 可隐藏 |
| 领用日期 | ❌ | 可显示 |
| 借用日期 | ❌ | 可显示 |
| 预计归还日期 | ❌ | 可显示 |
| 资产原值 | ❌ | 可显示 |
| 操作 | ✅ | 固定在右侧 |

#### 移动端卡片可配置字段

移动端使用卡片布局，支持配置卡片内显示的字段优先级：

| 优先级 | 字段 | 说明 |
|-------|------|------|
| P0（始终显示） | 资产照片、资产编码、资产名称 | 核心标识 |
| P1（默认显示） | 关系类型、资产状态、存放地点 | 关键信息 |
| P2（点击展开） | 规格型号、保管人、领用日期 | 次要信息 |
| P3（详情页） | 资产原值、采购日期、供应商 | 详细信息 |

## 6. 数据获取策略

### 6.1 预取策略

| 时机 | 预取内容 | 存储位置 |
|------|---------|---------|
| 移动端首页加载 | 用户资产概览（前50项） | IndexedDB |
| 进入我的资产页面 | 全部资产基础列表 | IndexedDB |
| 点击资产详情 | 该资产完整信息 | IndexedDB |
| 网络恢复 | 同步本地变更到服务器 | - |
