## 一、产品概述

**GZEAMS (Hook Fixed Assets)** 是一款基于 **元数据驱动的低代码平台** 架构的企业级固定资产管理系统。系统采用 Django 5.0 + Vue 3 技术栈，实现了从资产采购入库到报废处置的全生命周期管理，支持多组织数据隔离、可视化工作流设计、移动端盘点、第三方ERP集成等企业级功能。

**核心设计理念**：
- 元数据驱动 - 通过配置而非编码实现业务扩展
- 多组织隔离 - SaaS级的数据隔离架构
- 移动优先 - 支持离线盘点和移动审批
- 流程可视化 - LogicFlow可视化工作流设计器

**参考基准**：[NIIMBOT 钩子固定资产](https://yzcweixin.niimbot.com/)

## 二、核心功能模块

### 2.1 基础平台能力

| 功能 | 描述 |
|------|------|
| **多组织架构** | 支持集团→公司→分公司的三级组织结构，各部门间数据完全隔离 |
| **统一认证** | 企业微信/钉钉/飞书 SSO单点登录，自动同步组织架构 |
| **低代码引擎** | 17种字段类型，可视化表单设计，无需代码即可创建新业务单据 |
| **权限系统** | 字段级权限（读/写/隐藏/脱敏）+ 数据级权限（全部/本部门/本人） |
| **消息通知** | 7种通知渠道（站内信/邮件/短信/企微/钉钉/飞书/Webhook） |

### 2.2 资产全生命周期管理

```
采购申请 → 采购中 → 验收入库 → 在用 → 闲置 → 维修中 → 待报废 → 已报废
```

| 阶段 | 功能 |
|------|------|
| **采购申请** | 员工发起 → 多级审批 → 推送M18采购系统 → 实时同步进度 |
| **验收入库** | 质检 → 数量核对 → 自动生成资产卡片 → 打印二维码标签 |
| **日常操作** | 领用、调拨、借用、归还，均支持审批流程 |
| **维护保养** | 故障报修、计划性保养、维护成本统计 |
| **处置报废** | 报废/出售/捐赠/转移/销毁，技术鉴定 → 多级审批 → 生成财务凭证 |

### 2.3 资产分类体系

- **国家标准对齐**：符合 GB/T 14885 国家标准
- **多级分类树**：支持3级以上分类层级
- **折旧配置**：每类可配置折旧方法和预计使用年限
- **预设分类**：土地房屋、通用设备、专用设备、文物陈列品、图书档案、家具用具

### 2.4 耗材管理

| 特性 | 说明 |
|------|------|
| **与固定资产区别** | 价值通常<2000元，按批次/数量管理，领用即消耗 |
| **库存预警** | 正常/需补货/库存不足/超库存 四级预警 |
| **出入库管理** | 采购入库、领用出库、调拨、盘点 |
| **预设分类** | 办公用品、劳保用品 |

### 2.5 工作流引擎

| 功能 | 描述 |
|------|------|
| **可视化设计** | 基于LogicFlow的拖拽式流程设计器 |
| **节点类型** | 开始/结束、审批、条件分支、并行网关、抄送/通知 |
| **审批方式** | 或签（任一人）、会签（全部）、顺序签 |
| **超时处理** | 自动通过/自动驳回/超时提醒 |
| **字段权限** | 每个节点可配置字段的可编辑/只读/隐藏 |
| **版本控制** | 流程发布自动版本管理，支持回滚 |

### 2.6 盘点管理系统

| 功能 | 描述 |
|------|------|
| **盘点类型** | 全盘、部门盘、分类盘、抽查（N%） |
| **盘点方式** | 二维码扫描（逐个）、RFID批量扫描 |
| **离线支持** | IndexedDB本地存储，网络恢复后自动同步 |
| **快照机制** | 盘点开始时生成资产快照，确保数据一致性 |
| **差异处理** | 自动识别盘亏/盘盈/位置不符/保管人不符，支持审批调整 |

### 2.7 移动端能力

| 功能 | 描述 |
|------|------|
| **离线盘点** | 支持无网络环境下扫描盘点 |
| **移动审批** | 可视化流程图，批量审批，审批委托 |
| **数据同步** | 版本向量同步，冲突自动检测（服务器优先/客户端优先/合并/手动） |
| **安全机制** | AES-256加密、设备绑定、生物识别 |

### 2.8 第三方集成

| 集成类型 | 描述 |
|---------|------|
| **SSO集成** | 企业微信/钉钉/飞书 单点登录 + 组织架构同步 |
| **ERP集成** | M18系统采购单推送、财务凭证生成 |
| **财务集成** | 折旧计算、资产价值变动同步财务系统 |
| **数据导入导出** | Excel/CSV/PDF格式，支持异步大数据量处理 |

### 2.9 员工自助门户

| 模块 | 功能 |
|------|------|
| **我的资产** | 保管资产、使用资产、借用资产三合一视图 |
| **我的申请** | 领用/调拨/借用/退库申请及审批记录 |
| **我的任务** | 待审批、待处理、待盘点、待确认任务中心 |

## 四、应用场景

### 4.1 场景一：中大型企业固定资产管理

**适用对象**：500人以上、多部门、多地点的企业

**痛点**：
- 资产分散在各地点，管理混乱
- 谁在用什么资产不清楚
- 资产流失、闲置浪费严重
- 盘点耗时长、数据不准确

**解决方案**：
```
1. 组织架构映射 → 部门数据隔离
2. 二维码标签 → 一物一码精准管理
3. 资产领用流程 → 责任到人可追溯
4. 移动端盘点 → 效率提升80%
5. 差异预警 → 及时发现资产异常
```

### 4.2 场景二：集团型企业多组织管理

**适用对象**：集团公司、连锁企业

**痛点**：
- 各子公司系统各自为政
- 数据无法统一汇总分析
- 跨公司资产调拨困难

**解决方案**：
```
1. 三级组织架构 → 集团/公司/分公司
2. 数据隔离 + 跨组织调拨审批
3. 统一资产分类和编码规范
4. 集团级报表自动汇总
```

### 4.3 场景三：政府机关/事业单位

**适用对象**：政府机关、学校、医院

**痛点**：
- 国有资产监管要求严格
- 采购审批流程复杂
- 报废处置需多方审批
- 审计要求完整的数据追溯

**解决方案**：
```
1. 符合国标的分类体系
2. 可配置的多级审批流程
3. 全生命周期操作记录
4. 完整的审计日志
5. 财务系统自动对接
```

### 4.4 场景四：制造型企业设备管理

**适用对象**：制造业、工厂

**痛点**：
- 设备维保计划管理难
- 故障维修响应慢
- 设备利用率数据缺失

**解决方案**：
```
1. 计划性保养 → 提前预警自动派单
2. 故障报修 → 快速响应处理跟踪
3. 维保成本统计 → 设备投资回报分析
4. 设备状态监控 → 利用率优化
```

### 4.5 场景五：高值耗材管理

**适用对象**：实验室、医院、研发机构

**痛点**：
- 耗材价值高但数量大
- 领用频繁、库存难掌握
- 成本核算需要精确数据

**解决方案**：
```
1. 耗材独立模块管理
2. 四级库存预警
3. 领用即出库的成本核算
4. 批次效期管理
```

### 4.6 场景六：混合办公环境

**适用对象**：远程办公、多地办公的企业

**痛点**：
- 资产带回家无法追踪
- 跨地点转移记录缺失
- IT设备管理混乱

**解决方案**：
```
1. 借用流程 → 明确借用期限和责任
2. 移动端操作 → 随时随地办理手续
3. 位置变更记录 → 完整的转移轨迹
4. 到期提醒 → 自动通知归还
```

## 八、总结

GZEAMS是一个功能完整、架构先进的企业级固定资产管理平台：

1. **功能完整**：覆盖资产从申请采购到报废处置的全生命周期
2. **架构先进**：元数据驱动的低代码架构，灵活可扩展
3. **技术领先**：移动优先、离线支持、多组织隔离、可视化流程
