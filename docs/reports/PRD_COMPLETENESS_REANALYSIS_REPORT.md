### 4.3 Given-When-Then格式使用

**使用Given-When-Then格式的文件**: 6/29 test.md (21%)

**phase1_1_asset_category/test.md** (示例):
```python
def test_create_root_category(self):
    """测试创建根分类"""
    # Given: 测试组织已创建
    org = Organization.objects.create(name='测试组织')

    # When: 创建根分类
    category = AssetCategory.objects.create(
        organization=org,
        code='TEST001',
        name='测试分类'
    )

    # Then: 验证分类创建成功
    self.assertEqual(category.code, 'TEST001')
    self.assertEqual(category.level, 0)
```

### B. 评分方法说明

**完整性评分计算**:

```
总体评分 = (功能需求 × 30%) + (非功能需求 × 20%) + (业务背景 × 15%) +
           (验收标准 × 15%) + (可追溯性 × 10%) + (API规范 × 10%)

功能需求 = (文档结构 + 业务场景 + 角色权限 + 公共模型 + 数据模型 + API设计) / 6
非功能需求 = (性能 + 安全 + 可用性 + 扩展性 + 兼容性) / 5
业务背景 = (功能概述 + 业务场景 + 价值主张 + 利益相关者) / 4
验收标准 = (功能验收 + 非功能验收 + DoD + 测试映射) / 4
可追溯性 = (需求ID + 需求映射 + 变更历史) / 3
API规范 = (响应格式 + 错误码 + 批量操作) / 3
```
